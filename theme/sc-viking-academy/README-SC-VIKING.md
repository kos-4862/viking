# SC Viking Theme Notes

This theme is a locally customized base built from the `Multi Sports` WordPress theme.

## What changed

- The homepage template was rebuilt around a Juventus-style academy structure.
- The branding direction was changed toward `SC Viking`.
- The header and footer were simplified for a club enrollment flow.
- Typography was switched to `Montserrat` + `Bebas Neue`.

## First setup in WordPress

1. Install the theme folder `sc-viking-academy` into `wp-content/themes/`.
2. Activate the theme in WordPress admin.
3. Create a page for the homepage, for example `Home`.
4. Assign the page template `SC Viking Home`.
5. Set that page as the static front page in `Settings -> Reading`.
6. Set the site title to `SC Viking`.
7. Upload a custom logo.
8. Add a featured image to the homepage if you want the hero media card to use a real image.
9. Create and assign the primary menu.
